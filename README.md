# ts-utility-plugins

好玩、有趣、优雅的插件库

## Desc

插件库包括 `canvas特效`插件, `业务`插件、`工具组件`等等, 少数已标注的插件可用作生产环境, 其它仅供学习参考.

欢迎[贡献代码](#contributes)!

## Usage

### Step 1

- 进入对应[Bucket](#contributor)

### step 2

- 观阅`README`, 查看对应的插件介绍

### Step 3

- 于[项目dist文件夹](https://github.com/ddzy/ts-utility-plugins/dist)下找到同名`Bucket`的对应插件.
- 在项目中引入即可

> **PS**: 重构代码中, 部分导入功能暂未实现

#### Implemented

- `script`导入

#### Nonimplemented

- cdn
- `npm`导入
- `react`组件
- `vue`组件
